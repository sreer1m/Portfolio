# Editing Gallery Folder

Place your images here. They will be displayed in the gallery on the Editing portfolio page.

## How to add images:

1. Drop your image files here (e.g., `photo1.jpg`, `video-thumb1.png`)
2. Open `portfolio-editing.html`
3. Uncomment or copy a `<div class="gallery-item">` block
4. Set `src="editing-gallery/your-image.jpg"`
5. Update the caption, alt text, and category

## Supported formats:
- `.jpg`, `.jpeg`, `.png`, `.webp`
- Use optimized/compressed images for faster loading
- Recommended: at least 1200px wide for high-quality lightbox view
