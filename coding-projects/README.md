# Coding Projects Folder

Place your HTML project files here. Each file will be embedded as a live preview on the Programming portfolio page.

## How to add a project:

1. Drop your HTML file here (e.g., `calculator.html`, `todo-app.html`)
2. Open `portfolio-coding.html`
3. Uncomment or copy a `<div class="coding-project-card">` block
4. Set `data-src="coding-projects/your-file.html"`
5. Update the title, description, and tech tags

## Supported formats:
- `.html` files (embedded as live preview in iframe)
- For Python/other projects, link to GitHub instead
