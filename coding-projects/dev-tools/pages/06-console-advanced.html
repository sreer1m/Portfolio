🚀 Advanced Console Techniques

$ Shortcuts - Quick Selectors

The Console has special shortcuts that make selecting elements faster than typing out the full commands!

$() - The Quick Selector

Instead of typing document.querySelector(), just use $()

// Old way:
document.querySelector('h1')

// Quick way:
$('h1')  // Same result!

$$() - Select All Elements

Instead of document.querySelectorAll(), use $$()

// Get all buttons on the page:
$$('button')

// Count all paragraphs:
$$('p').length

$0 - The Last Selected Element

$0 always refers to the last element you clicked on in the Elements tab!

// Click any element in Elements tab, then type:
$0  // Shows that element

$0.style.background = 'yellow'  // Highlight it!

Console Styling - Make Messages Beautiful

You can add CSS styling to your console messages using %c!

console.log('%cHello!', 'color: blue; font-size: 20px;');

console.log('%cError!', 'color: red; font-weight: bold; background: yellow;');

💡 Pro Tip

Many websites use styled console messages to show warnings or easter eggs. Check the console on popular websites like Facebook or YouTube!

Measuring Code Performance

Want to know how fast your code runs? Use console timing functions!

console.time() & console.timeEnd()

console.time('My Process');
// ... do something ...
for(let i = 0; i < 1000000; i++) {
    // some work
}
console.timeEnd('My Process');
// Output: "My Process: 12.5ms"

Grouping Console Messages

Organize related messages together with console.group()!

console.group('User Information');
console.log('Name: Alice');
console.log('Age: 25');
console.log('City: New York');
console.groupEnd();

console.group('System Information');
console.log('OS: Windows');
console.log('Browser: Chrome');
console.groupEnd();

console.count() - Tracking Function Calls

Count how many times something happens:

function doSomething() {
    console.count('Function called');
}

doSomething();  // "Function called: 1"
doSomething();  // "Function called: 2"
doSomething();  // "Function called: 3"

console.assert() - Testing Conditions

Only log a message if something is wrong:

const age = 15;
console.assert(age >= 18, 'User is not old enough!');
// Only shows error if age < 18

const isLoggedIn = true;
console.assert(isLoggedIn, 'User must be logged in!');
// No error because condition is true

Copying from Console

You can copy data from the console to your clipboard!

// Get all links on page
const links = $$('a').map(a => a.href);

// Copy to clipboard
copy(links);

// Now paste (Ctrl+V) anywhere!

After clicking, paste (Ctrl+V) into a text editor to see the copied data!

Live Expressions

Create live expressions that constantly update! Look for the "eye" icon in the Console.

How to Create a Live Expression:

  1. Open the Console
  2. Click the "eye" icon (👁️) near the top
  3. Type an expression like document.body.scrollTop
  4. Scroll the page and watch it update in real-time!

Useful Live Expressions:

  • window.innerWidth - Watch window width as you resize
  • document.body.scrollTop - See scroll position
  • $$('button').length - Count elements dynamically
  • new Date().toLocaleTimeString() - Live clock

Key Takeaways

  • ✅ Use $() and $$() as shortcuts for selecting elements
  • ✅ $0 refers to the last clicked element in Elements tab
  • ✅ Style console messages with %c for better visibility
  • ✅ Use console.time() to measure performance
  • ✅ Group related messages with console.group()
  • ✅ Use console.count() to track how many times things happen
  • ✅ Use copy() to copy data from console to clipboard
  • ✅ Live expressions update automatically - great for monitoring!
