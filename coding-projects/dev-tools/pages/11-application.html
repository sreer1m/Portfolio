📦 Application Tab

What is the Application Tab?

The Application tab shows data stored by the browser such as cookies, localStorage, sessionStorage, and cache.

Storage Demo

Inspect This

  1. Open DevTools and go to Application
  2. Click Local Storage
  3. Select this site
  4. Watch values appear and disappear

localStorage stays even after refreshing the page. sessionStorage is cleared when the tab closes.
