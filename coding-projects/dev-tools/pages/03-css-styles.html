🎨 CSS Styles Panel: Understanding & Editing Styles

What Is CSS?

CSS (Cascading Style Sheets) is what makes websites look beautiful. While HTML provides the structure (skeleton), CSS provides the style (skin, colors, layout).

Think of it this way:

  • HTML = The words in a book
  • CSS = The fonts, colors, spacing, and layout

🎯 What CSS Controls:

  • Colors: text color, background color
  • Typography: font family, size, weight
  • Spacing: margins, padding
  • Layout: positioning, width, height
  • Effects: shadows, borders, animations
  • Responsive design: how things look on different screens

The Styles Panel in DevTools

When you inspect an element in the Elements tab, you'll see a panel on the right (or below) called Styles. This panel shows ALL the CSS rules affecting that element.

Anatomy of a CSS Rule

.selector {
    property: value;
    color: blue;
    font-size: 16px;
}
  • Selector - What element(s) the rule applies to (.classname, #id, tagname)
  • Property - What aspect to style (color, font-size, padding, etc.)
  • Value - The specific style (blue, 16px, center, etc.)

Understanding CSS Priority

Multiple CSS rules can affect the same element. DevTools shows them in order of priority (specificity):

1. Inline styles (style="color: red") - HIGHEST PRIORITY

2. ID selectors (#myId)

3. Class selectors (.myClass)

4. Element selectors (div, p) - LOWEST PRIORITY

🔑 Key Concept: The Cascade

If the same property is defined multiple times, the more specific or later-defined rule wins. Strikethrough text in DevTools means that rule was overridden by a more specific one.

Adding New CSS Properties

You can add entirely new CSS properties in DevTools!

🎮 Add Your Own Styles
Add Custom Styles to Me!

📋 Add New Properties:

  1. Right-click the blue box above and inspect it
  2. In the Styles panel, find element.style
  3. Click in the empty space inside the rule block
  4. Type a new property, like: border: 5px solid yellow
  5. Press Enter and watch it apply!
  6. Try adding: transform: rotate(5deg)
  7. Try: box-shadow: 0 10px 20px rgba(0,0,0,0.3)

💡 Pro Tip:

DevTools has autocomplete! Start typing a property name and it will suggest options. This is great for discovering CSS properties you didn't know existed.

Computed Styles Tab

Next to the Styles panel, there's a Computed tab. This shows the final values for ALL CSS properties, after all cascading and inheritance.

Check my computed styles!

📋 View Computed Styles:

  1. Inspect the box above
  2. Click the Computed tab (next to Styles)
  3. You'll see a HUGE list of properties
  4. These are ALL the CSS properties with their final computed values
  5. Try searching for "font-size" in the filter box
  6. Click the arrow next to a property to see where it came from

🔑 When to Use Computed:

Use the Computed tab when you need to know the exact final value of a property, especially for things like width, height, or inherited properties. It's perfect for debugging layout issues.

Common CSS Properties to Know

Property What It Does Example Values
color Text color red, #ff0000, rgb(255,0,0)
background-color Background color blue, #0000ff, rgba(0,0,255,0.5)
font-size Text size 16px, 1.5em, 120%
font-weight Text thickness normal, bold, 700
padding Space inside element 10px, 20px 10px
margin Space outside element 15px, 10px 20px 10px 20px
border Border around element 1px solid black
border-radius Rounded corners 5px, 50% (circle)
width / height Element size 200px, 50%, auto
display How element displays block, inline, flex, none

Key Takeaways

  • CSS controls all visual styling (colors, fonts, spacing, layout)
  • The Styles panel shows all CSS rules affecting an element
  • You can edit any CSS property by clicking on it in DevTools
  • You can add new properties by clicking in empty space in a rule
  • CSS has a cascade more specific rules override less specific ones
  • Strikethrough text means that rule was overridden
  • The Computed tab shows final calculated values for all properties
  • You can disable properties by unchecking the checkbox
